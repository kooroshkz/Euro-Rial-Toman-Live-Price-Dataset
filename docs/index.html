About This Dataset

This dataset provides comprehensive historical exchange rate data for Eueo to Iranian Rial (IRR) from 2011 to the present day.

  • Source: TGJU.org (Tehran Gold and Jewelry Union)
  • Update Frequency: Daily via automated scraping
  • Data Points: Open, Low, High, Close prices, Change Amount, Change Percent, Dates
  • Currency: Rial (IRR)
